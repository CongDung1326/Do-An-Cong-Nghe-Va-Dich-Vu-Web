# Tên Sản Phẩm
Website hỗ trợ bán tài khoản game Liên Minh Huyền Thoại

## Tính Năng
- Phân trang giữa admin và user
- Bảo mật thông tin người dùng và tải khoản game
- Giao diện dễ dàng sử dụng
- Có hỗ trợ admin trong việc quản lý website
